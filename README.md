# Whitehouse Petitions

Project created for the purpose of learning Swift language. This project shows data from an online JSON file with the demands to Whitehouse made by USA citizen.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

* Xcode 11 installed. Download and install it from: [Xcode](https://apps.apple.com/es/app/xcode/id497799835?mt=12)

### Installing

1. Download or clone the repository into a folder in your local drive.
2. Locate the file named **Whitehouse Petitions.xcodeproj** and click on it to open the Xcode IDE. 

## Deployment

* Hit Cmd + R and the app will be loaded on the simulator.
